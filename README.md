Ouvrez votre navigateur ou utilisez un outil comme Postman, Insomnia, ou curl pour tester vos endpoints :  

Récupérer toutes les villes : http://localhost:3000/api/cities  
Récupérer une ville par ID (le premier ID sera 1) : http://localhost:3000/api/cities/1  
Rechercher des villes (par exemple, "paris") : http://localhost:3000/api/cities?search=paris
